## Step 1 — Cross-Reference Memory

For each client mentioned in the notes, query `memory` MCP:
- Retrieve any recurring themes, longstanding concerns, or open actions from prior weeks
- Note whether this week's interaction progressed, resolved, or added to those recurring items
- If a client appears in memory with unresolved prior actions, flag them in the "Carried-Over Actions" section
